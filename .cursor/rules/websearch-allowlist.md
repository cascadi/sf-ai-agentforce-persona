# Web Search Domain Allowlist

When using WebSearch or WebFetch, prefer results from and freely access the following Salesforce-owned domains. These are trusted first-party sources.

- `force.com` — Platform, Visualforce, custom domains
- `lightning.force.com` — Lightning Experience
- `developer.salesforce.com` — Developer docs, Trailhead modules, API references
- `help.salesforce.com` — Help & Training
- `trailhead.salesforce.com` — Trailhead learning platform
- `admin.salesforce.com` — Admin blog
- `architect.salesforce.com` — Architect resources
